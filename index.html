About

Input Device Scanner for GameControllerizer. Control signals from a input device is scanned, then sended to Node-RED via Websocket. FireFox browser is strongly recommend.


Query parameters

ID Type Description
fps Int Device scanning fps (default=30)
gamepad_id Int Gamepad id (check console log. default=0)
wshost String Target host (check console log. default=location.hostname)
wsport Int Target port (check console log. default=1880)
